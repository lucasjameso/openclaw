## Goal

Do not make Vince into a generic AI spokesman.

Vince should feel like:
- relaxed
- competent
- mildly amused
- not corporate
- not robotic
- like he has personally watched the machine do dumb shit all week

## Shot Pattern

Keep it simple:

1. face / avatar cold open
2. stat card cut-in
3. one diagram or dashboard plate
4. back to avatar for the punchline
5. end card CTA

Avoid over-editing. Let the voice and the absurdity do the work.

## Background / Brand Rules

- dark Forge background
- no fake office
- no white seamless studio
- no generic tech gradient
- use dashboard blue + amber accents if overlays are added

## Avatar Rules

- same avatar every time
- same clothing vibe every time
- same calm, slightly amused expression
- no over-animated gestures
- if the avatar feels too polished, it is wrong

## CTA Rules

End every Vince video with one of:
- `Follow @Forge_Builds`
- `Watch the machine build in public`
- `Read the $809 Mistake Guide`

Do not stack 3 CTAs in one video.

## Best Sequence For Morning

1. Generate Vince intro
2. Generate Vince $75/day leak
3. Generate Vince speed-running mediocrity
4. Pair each with one static card or Excalidraw cut-in
5. Publish one, hold two in reserve
