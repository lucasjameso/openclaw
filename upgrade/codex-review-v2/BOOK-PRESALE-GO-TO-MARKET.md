# Book Pre-Sale Go-To-Market

Updated: March 26, 2026

This is the practical plan for pushing the book next week without wasting time on launch theater.

## Core Decision

If pre-sales need to start in the week of March 30, 2026, the fastest route is:

1. use Gumroad or Kit Commerce for direct pre-sales now
2. use Kit landing pages for email capture and launch messaging
3. use Amazon KDP only if the eBook manuscript is genuinely stable enough for a preorder

Reason:
- Gumroad and Kit let us sell a digital product immediately
- KDP preorder has real penalties if we miss deadlines or cancel
- KDP print does **not** support pre-orders

## Best Launch Stack

### Option A: Fastest and safest

- Product checkout: Gumroad
- Email capture + prelaunch page: Kit
- Traffic: X, story page, dashboard screenshots, Roger/Vince clips

Why this is best:
- fastest to ship
- lowest operational risk
- easiest to iterate on copy, pricing, and bonuses

### Option B: If we want all-in-one creator stack

- Product checkout: Kit Commerce
- Email capture: Kit landing page
- Growth: Creator Network + X + direct links

Why:
- one system for leads and sales
- easier follow-up automation later

### Option C: Amazon preorder

Use only if:
- eBook manuscript is close enough to final that missing the date is very unlikely
- we want Amazon detail-page credibility more than speed

Do **not** use this as the first move if the book is still changing heavily.

## What To Sell

Do not just sell “a book.”

Sell:
- the core book
- a clear promise
- a bonus reason to buy now

Recommended initial offer:
- `The $809 Mistake Guide`
- price: `$19`
- bonus for pre-order buyers:
  - launch notes / update email
  - one bonus checklist
  - one behind-the-scenes artifact pack

This makes the pre-sale feel like a live experiment, not a static PDF.

## Positioning

The strongest hooks are not “AI book” hooks.

They are:
- `$809 spent to learn what breaks`
- `$75/day leak fixed to ~$1.60/day`
- `we built the machine, then had to survive it`
- `buy the version with the mistakes already paid for`

Best direct positioning line:

`Spend $19 instead of spending $809 making the same mistakes yourself.`

## Half-Day Marketing Sprint

This is the half-day block to run before the pre-sale page goes live.

### Block 1: Offer clarity (45 minutes)

Decide:
- title
- subtitle
- promise
- price
- pre-order bonus
- exact CTA

Output:
- one product headline
- one product subtitle
- three bullets
- one CTA

### Block 2: Sales page assets (60 minutes)

Need:
- final cover
- top 3 chapter cards
- launch announcement hero
- one dashboard screenshot
- one funny short-form cover

Output:
- all assets linked from one folder

### Block 3: Launch copy (45 minutes)

Write:
- one X launch post
- one thread opener
- one email to the list
- one short “why this exists” paragraph for the product page

### Block 4: Funnel wiring (45 minutes)

Set up:
- Kit landing page
- Gumroad or Kit product
- confirmation email
- post-purchase follow-up note
- tracking link source tags

### Block 5: Test purchase + friction pass (30 minutes)

Check:
- checkout flow
- file delivery
- mobile view
- send-to-Kindle option if using Gumroad
- email delivery

## The Actual Funnel

Best simple funnel:

1. X post / clip / screenshot
2. landing page or product page
3. purchase or email signup
4. follow-up email with story + reminder
5. second launch post with receipts

Do not overcomplicate this.

The first goal is:

## Platform Notes

### Amazon KDP

- Kindle eBook pre-orders can be set up up to one year in advance
- if you miss the preorder manuscript deadline or cancel, you can lose preorder access for one year
- print books on KDP do not support pre-orders

### Kit

- landing pages can live on a hosted Kit URL or a custom domain
- Kit can sell digital products directly
- Kit Creator Network can help grow email subscribers

### Gumroad

- good for immediate digital sales
- enable test purchases before launch
- enable purchasing power parity if global affordability matters
- enable tipping if we want upside from supporters
- Gumroad can send eligible purchased files to Kindle if file type and size fit their rules

## Recommendation

For next week:

1. launch pre-sales through Gumroad or Kit first
2. use Kit landing pages for signup and email capture
3. use KDP later, only when the eBook is stable enough that deadline risk is low

That gets us speed, control, and a lower chance of stepping on a platform penalty.
